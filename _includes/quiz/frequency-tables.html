📌 Try It Yourself

Q: You have a list of 100 product categories (e.g., Electronics, Clothing, Books). What kind of chart and table should you use to explore the data?

💡 Show Answer

✅ Use a frequency table to count how many times each category appears, and a bar chart to visualize it — because the data is categorical.

Bonus: What if you had 100 values showing product prices?

💡 Show Answer

✅ Use a frequency table with **intervals** (e.g., 0–50, 50–100) and a histogram — since the data is numerical and continuous.
