Q: You’re given a table of patient data with the following columns:

  • 🆔 Patient ID
  • 🎂 Age
  • 🩺 Blood Type
  • 📍 City
  • 🧪 Cholesterol Level

Which of the following is a case?

💡 Show Answer

✅ A row in the table — like a single patient’s full record.
Each case is one unit of observation (e.g. one patient), and each variable is a column describing something about them (age, city, etc.).
