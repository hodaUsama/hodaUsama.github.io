Q: You’re given a table of patient data with the following columns:

  • 🆔 Patient ID
  • 🎂 Age
  • 🩺 Blood Type
  • 📍 City
  • 🧪 Cholesterol Level

What would be considered a single case in this dataset?

💡 Show Answer

✅ A row in the table — that is, one patient’s full record.

Each case represents one unit of observation (like a patient), while each variable is a column describing something about that case (e.g., age, blood type, city).
